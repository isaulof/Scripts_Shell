# Scripts_Shell
This repository contains a collection of open-source Shell scripts designed for automation, system administration, and general utility tasks. Feel free to use, download, or contribute your own scripts!

Este repositório contém uma coleção de scripts Shell de código aberto voltados para automação, administração de sistemas e tarefas utilitárias em geral. Sinta-se à vontade para usar, baixar ou contribuir com seus próprios scripts!

Guarde o script na sua pasta BIN. Depois cole o caminho do arquivo no arquivo de configuração bash. Depois aplique as permissões de execução ao script.sh
